Icon 图标的实现，首选 css，其次svg，再其次使用 iconfont，便于引入后覆盖

并需要提供方法实现 系统级 Icon，以及项目级、页面级 Icon 的定制

- 系统级提供常用图标共几大类
  - 操作反馈提示
  - 常见标识
- 项目级自己项目内维护，亦作为公共图标
- 页面级不作要求（临时图标，甚至可使用图片等，生命周期简单或过于复杂的图片，不作为 icon 考量）
